Simple Frogger
March 4th, 2021

Hello!

I was poking around some old Unity files on my computer and I totally forgot that I had started a simple 2D project a few months ago. It was about halfway done, so I went ahead and took a few days to finish it up. It's just a simple frogger game, but I had fun going back and completing it because it's the perfect kind of small project that I can use to get back on the proverbial horse. I uploaded it to my portfolio if you want to check it out!

MacOS builds have been hurting my head lately, so I am going to address that later, but for now you can play the game on Windows or on WebGL (i.e. in-browser). It seems to me there aren't a ton of web browsers that support client-side WebGL games, but if you have one then go for it :)

(Others might describe this as minimal graphics, but I like to think of it as a complex work in style of the De Stijl art movement)

--Josh
